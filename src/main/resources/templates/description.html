    • Historias de Usuario para Usuario Votante

    HUV.1. Creación de cuenta

    Como un usuario votante nuevo, quiero poder crear una cuenta en la página web de música para acceder a todas las características y funcionalidades.

    Criterios de aceptación:

    • El usuario debe proporcionar un nombre de usuario válido, una dirección de correo electrónico y una contraseña segura al registrarse.
    • Después de la creación exitosa de la cuenta, debo recibir un correo electrónico de confirmación.

    HUV.2. Activación de cuenta

    Como usuario votante registrado, quiero poder activar mi cuenta haciendo clic en el enlace de confirmación enviado a mi dirección de correo electrónico.

    Criterios de aceptación:

    • Debe generarse un enlace de activación único y enviado al correo electrónico proporcionado durante la creación de la cuenta.
    • Al hacer clic en el enlace de activación, mi cuenta debe activarse y poder iniciar sesión.

    HUV.3. Autenticación de usuario

    Como usuario votante, quiero poder iniciar sesión en mi cuenta para acceder a las funciones personalizadas y votar por listas de reproducción.

    Criterios de aceptación:

    • Debo poder ingresar mi nombre de usuario y contraseña en el formulario de inicio de sesión.
    • Después de iniciar sesión correctamente, debo ser redirigido a mi página de perfil personalizada.

    HUV.4. Ver listas por género musical creadas

    Como usuario votante, quiero poder ver las listas de reproducción creadas por otros usuarios, filtradas por género musical, para descubrir nueva música.

    Criterios de aceptación:

    • Debe haber una sección de búsqueda y navegación por género en la página de inicio.
    • Al seleccionar un género, debo ver una lista de listas de reproducción creadas por otros usuarios para ese género específico.

    HUV.5. Buscar una canción

    Como usuario votante, quiero tener la capacidad de buscar una canción específica para encontrar las listas de reproducción en las que está incluida.

    Criterios de aceptación:

    • Debe haber un campo de búsqueda en la página de inicio.
    • Al ingresar el nombre de una canción, debo ver una lista de listas de reproducción que contienen esa canción.

    HUV.6. Votar/Desvotar por una lista

    Como usuario votante, quiero tener la opción de votar positiva o negativamente por una lista de reproducción para expresar mi opinión sobre su contenido.

    Criterios de aceptación:

    • En cada lista de reproducción, debe haber botones distintos para votar positivamente y negativamente.
    • Solo puedo votar una vez por cada lista de reproducción.
    • Debo poder cambiar mi voto de positivo a negativo y viceversa si cambio de opinión.

    Historias de Usuario para Usuario Administrador

    HUV.1. Autenticación de usuario

    Como administrador, quiero poder iniciar sesión en mi cuenta de administrador para acceder a las herramientas de gestión del sitio.

    Criterios de aceptación:

    • Debo ingresar las credenciales de administrador en el formulario de inicio de sesión.
    • Después de iniciar sesión, debo ser redirigido a un panel de control administrativo.

    HUV.2. CRUD Géneros musicales

    Como administrador, quiero poder realizar operaciones CRUD (Crear, Leer, Actualizar, Eliminar) en la lista de géneros musicales disponibles en la página.

    Criterios de aceptación:

    • Debo tener acceso a una sección de administración de géneros musicales en el panel de control.
    • Puedo crear nuevos géneros musicales, editar sus nombres y descripciones, ver una lista de todos los géneros existentes y eliminar géneros no deseados.

    HUV.3. CRUD Canciones

    Como administrador, quiero poder realizar operaciones CRUD en la lista de canciones disponibles en la página.

    Criterios de aceptación:

    • Debo tener acceso a una sección de administración de canciones en el panel de control.
    • Puedo agregar nuevas canciones proporcionando información como título, artista, género y enlace.
    • Puedo editar la información de las canciones existentes, ver una lista de todas las canciones y eliminar canciones si es necesario.
